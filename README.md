# Apple Stock Price Analysis & Prediction

## Overview
This project analyzes historical Apple (AAPL) stock price data and performs time series analysis to understand trends and predict future price movements.

## Key Analysis
- Stock price trend visualization
- Moving averages (SMA/EMA)
- Volatility analysis
- Correlation analysis
- Price prediction model

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Outcome
The project demonstrates financial data analysis skills and time-series forecasting concepts.
